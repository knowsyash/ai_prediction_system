# Literature Review

## Existing Work
* Basic lexicon-based sentiment analysis.
* Dashboards showing only historical review stats.
* Standard e-commerce web scraping tools.

## Key Findings
* Reviews heavily dictate electronics purchasing behavior.
* Deep Learning (BERT) captures nuanced sentiment better than traditional methods (VADER).
* Rating trends strongly correlate with long-term sales.

## Research Gap
* **Single Model Bias:** Lack of robust ensemble techniques (Soft Voting, Bagging).
* **No Forecasting:** Systems analyze past data but fail to predict *future* ratings.
* **Poor Data Quality:** Automated systems struggle to properly clean messy real-world text.
* **Fragmented Pipelines:** No end-to-end integration mapping scraping to predictive analytics.
