# Introduction

## Project Overview
An end-to-end AI pipeline that extracts e-commerce reviews (iPhone 15, iPhone 16, iQOO Z10) and applies NLP and Machine Learning to gauge sentiment, forecast future ratings, and power an interactive business dashboard.

## Project Objective
* **Automate Data Pipeline:** Scrape and clean raw smartphone reviews.
* **Analyze Sentiment:** Measure customer satisfaction using advanced NLP models.
* **Aggregate Scores:** Use ensemble modeling to track accurate reliability trends.
* **Predict Ratings:** Forecast next month's product performance.
* **Deliver Insights:** Provide an interactive dashboard for data-driven decisions.
